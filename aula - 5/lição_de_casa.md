
Os operadores de atribuição são usados para atribuir valores a variáveis. O operador básico é o `=`, mas existem operadores compostos que facilitam operações matemáticas e de bits.

## Operador de Atribuição Simples (`=`)
```java
int x = 10; // atribui o valor 10 à variável x
````

## Soma com Atribuição (`+=`)

```java
int x = 10;
x += 5; // equivalente a x = x + 5
// resultado: 15
```

## Subtração com Atribuição (`-=`)

```java
int x = 10;
x -= 3; // equivalente a x = x - 3
// resultado: 7
```

## Multiplicação com Atribuição (`*=`)

```java
int x = 10;
x *= 2; // equivalente a x = x * 2
// resultado: 20
```

## Divisão com Atribuição (`/=`)

```java
int x = 10;
x /= 2; // equivalente a x = x / 2
// resultado: 5
```

## Módulo com Atribuição (`%=`)

```java
int x = 10;
x %= 3; // equivalente a x = x % 3
// resultado: 1 (resto da divisão)
```

## AND com Atribuição (`&=`)

```java
int x = 5;   // 0101
x &= 3;      // 0011
// resultado: 0001 (1)
```

## OR com Atribuição (`|=`)

```java
int x = 5;   // 0101
x |= 3;      // 0011
// resultado: 0111 (7)
```

## XOR com Atribuição (`^=`)

```java
int x = 5;   // 0101
x ^= 3;      // 0011
// resultado: 0110 (6)
```

## Deslocamento à esquerda (`<<=`)

```java
int x = 5;   // 0101
x <<= 1;
// resultado: 1010 (10)
```

```java
int x = 10;  // 1010
x >>= 1;
// resultado: 0101 (5)
```
